## Installation

### From source

```shell
git clone git@github.com:marianocarrazana/kubernety.git
cd kubernety
pip install -e .
# update only:
git pull
```

### From pip

```shell
pip install -U kubernety
```

## Usage

Run on your console:

```shell
kubernety
```
